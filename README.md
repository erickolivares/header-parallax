# Header-Parallax
Parallax Effect for background header images

##JS Include
```
<script src="parallax.js"></script>
```

###HTML Setup
target div must have "parallax" class

```
<section class="parallax">
</section>
```

###CSS Setup

```
.parallax {
  background-position: 50% 0%;
  background-size: cover;
}
```
